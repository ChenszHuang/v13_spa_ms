# SPA Management System

SPA Management System is a web-based application designed to help spa businesses manage daily operations such as customer records, therapist management, appointments, services, invoicing, and payments.

This system improves efficiency by centralizing spa operations into one platform, making it easier to track bookings, transactions, and customer history.

---

## Features

- Customer Management (Profiles, Contact Details, History)
- Therapist Management
- Service & Package Management
- Appointment / Booking Scheduling
- Invoice Generation
- Payment Tracking
- Reports & Analytics (Sales, Booking Summary, etc.)
- User Access Control (Admin / Staff roles)
